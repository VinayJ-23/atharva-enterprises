# Brand Images Setup Guide

## How to Add Brand-Specific Images

### Option 1: Use Local Images (Recommended)

1. **Create the brands folder:**
   ```
   public/brands/
   ```

2. **Download brand logos/bag images** from:
   - Official brand websites
   - Stock photo sites (Unsplash, Pexels, Pixabay)
   - Brand media kits

3. **Save images with these exact filenames:**

   **Cement Brands:**
   - `acc.png` - ACC Cement bag/logo
   - `vicat.png` - Vicat Cement bag/logo
   - `ambuja.png` - Ambuja Cement bag/logo
   - `bangur.png` - Bangur Cement bag/logo
   - `dalmia.png` - Dalmia Cement bag/logo
   - `jk.png` - JK Cement bag/logo
   - `zuari.png` - Zuari Cement bag/logo
   - `deccan.png` - Deccan Cement bag/logo
   - `nagarjuna.png` - Nagarjuna Cement bag/logo

   **Steel Brands:**
   - `tata.png` - TATA Steel logo/bars
   - `jsw.png` - JSW Steel logo/bars
   - `sail.png` - SAIL Steel logo/bars
   - `shreeom.png` - ShreeOm Steel logo/bars

   **Construction Materials:**
   - `aac-block.png` - AAC Block product image
   - `admixture.png` - Admixture product image
   - `plumbing.png` - Plumbing materials image
   - `white-cement.png` - White Cement bag image
   - `iso-certified.png` - ISO Certified badge
   - `primer.png` - Primer product image

4. **Image Requirements:**
   - Format: PNG or JPG
   - Recommended size: 400x500px or larger
   - Transparent background preferred for logos
   - High quality (at least 72 DPI)

### Option 2: Use Online Image URLs

The current code uses Unsplash images as placeholders. You can replace them with:
- Direct image URLs from brand websites
- CDN-hosted images
- Stock photo service URLs

### Where to Find Brand Images

1. **Official Brand Websites:**
   - Visit each brand's official website
   - Look for "Media Kit" or "Press Resources" section
   - Download official logos and product images

2. **Stock Photo Sites:**
   - Unsplash.com - Free high-quality photos
   - Pexels.com - Free stock photos
   - Pixabay.com - Free images
   - Search for: "cement bag", "steel bars", "construction materials"

3. **Logo Repositories:**
   - Logoshape.com
   - Logo.com
   - Brandfetch.com

### Quick Setup Script

Run the provided `download-brand-images.js` script to download placeholder images, then replace them with actual brand images.

### Legal Note

⚠️ **Important:** Ensure you have proper licensing/permissions to use brand logos and images on your website. Some brands require authorization for logo usage.
